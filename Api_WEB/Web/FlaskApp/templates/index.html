Explications rapides


- Consultez notre README (disponible sur GIT) pour plus de détails.

- Vous retrouvez sur les pages Carte et Analyse des analyses rapides liées aux élements.

- Nos données sont récupérées grâce à Scrapy, et sont stockées sur une base de données Mongo.

- Vous disposez d'un moteur de recherche (comme sur un site de commerce en ligne) sur la page Base de données, il repose sur ElasticSearch.
Celui-ci, vous permet d'afficher, selon plusieurs caractéristiques (en même temps), les films présents dans la base de données Mongo. Vous pouvez également trier les résultats par ordre croissant et décroissant (rang). De plus, nous avons mis en place un système de pages lorsqu'il y a trop de films. Vous pouvez naviguer librement grâce à un menu déroulant et/ou grâce à des boutons présents bas de page. Bonne exploration ٩(^ᴗ^)۶

- Vous disposez d'une visualisation géographique du nombre de films présents dans ce top par pays sur la page Carte.

- Enfin, vous disposez d'une analyse de données et de plus de détails sur ce qu'on peut apprendre sur la page Analyse !
